# 컴퍼니인포 소개
컴퍼니인포는 국내 상장사의 정보를 제공하는 웹사이트 입니다. 사업보고서를 제출할 의무가 있는 국내 모든 법인의 기업 정보는 전자공시시스템(DART)을 통해 공개되고 있습니다. 컴퍼니인포는 DART의 데이터를 검색할 수 있고 결과에 맞는 데이터를 시각화해서 제공합니다.
